# English

## awesome-ahakey

A curated collection of community projects, tools, workflows, demos, and showcases for AhaKey.

This repository is intended to highlight what the community is building around AhaKey, including:

- third-party clients
- tools and utilities
- workflow presets
- setup showcases
- tutorials and guides
- experiments and integrations

## Sections

### Clients
Community-built desktop clients, native clients, and alternative interfaces.

### Tools
Debugging tools, inspectors, helpers, converters, and other useful utilities.

### Workflows
Workflow presets, usage patterns, setup ideas, and coding flows.

### Showcases
Screenshots, desk setups, demos, videos, and community highlights.

### Tutorials
Articles, guides, setup notes, and onboarding materials.

### Integrations
Claude, Cursor, Codex, editor integrations, and other ecosystem connections.

## Contributing

If you built something around AhaKey, or found a useful community resource, feel free to open a pull request or share it in AhaKey Discussions.

# 简体中文

## awesome-ahakey

这里用于收录围绕 AhaKey 社区产生的项目、工具、workflow、演示和 showcase。

这个仓库的目标，是把社区里真正有价值、能被别人看见和复用的东西整理出来，包括：

- 第三方客户端
- 小工具和实用脚本
- workflow 预设
- setup showcase
- 教程和指南
- 各种实验性集成

## 收录分类

### Clients
社区开发的桌面客户端、原生客户端和替代界面。

### Tools
调试工具、检查工具、辅助脚本、转换工具等。

### Workflows
workflow 预设、使用方式、setup 灵感和 coding flow。

### Showcases
截图、桌面布局、演示视频和社区精选内容。

### Tutorials
文章、教程、安装笔记和上手指南。

### Integrations
Claude、Cursor、Codex、编辑器集成和其他生态连接。

## 参与方式

如果你围绕 AhaKey 做了项目，或者发现了值得整理的社区资源，欢迎提交 pull request，或者先发到 AhaKey Discussions。
